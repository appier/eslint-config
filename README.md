# Appier ESLint Config


## Getting Started


### Step 1: Installation

```sh
bun add -D eslint @appier/eslint-config typescript
```


### Step 2: Create Config

Create a `eslint.config.js` in the root.


```js
import { defineConfig } from '@appier/eslint-config'


export default defineConfig({
  //  your config
})
```
